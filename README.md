# Origin / Credits
This is the GUI Version of the [CS2 Inspect Generator](https://github.com/dr3fty/cs2-inspect-gen) from [dr3fty](https://github.com/dr3fty)

# Usage
- install Python 3    
- download the Zip  
- unpack
- run the pre_start.bat
- run the start.bat
